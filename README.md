Atividade final de módulo - Módulo introdução ao back-end.

Programa starter fullstack web developer 16ª edição.

Aplicação consiste em:

  ○ Criação de conta
  
  ○ Login
  
  ○ CRUD* de recados
  

*CRUD (create, read, update e delete)

Aplicação possui:

● Dados de um usuário:

  ○ Identificador
  
  ○ Nome
  
  ○ E-mail
  
  ○ Senha
  

● Dados de um recado:

  ○ Identificador
  
  ○ Título
  
  ○ Descrição
